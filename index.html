1.Inline>Internal>External
  • use style="background-color: color" after "body" to change color.(Inline CSS)
  • Within the "head" section, create a "style" tag. Be sure to follow standard CSS syntax, including selectors, properties, and values.(Internal CSS)
  • external CSS advantages over internal or inline CSS. HTML code is kept clean and uncluttered, makes it easier to maintain and update your styles, allows you to reuse styles across multiple pages.Add a link to the external CSS file in the section.

2.CSS syntax
  • CSS syntax:selector {property: value;}
  • use "class=name" to differentiate an html element, then use .name in css to make it a selector.n words inside class heading makes n classes.
  • use "id=name" same as class by using #name, diff. is just that class's name can be used any no. of times, id can't.
  • A pseudo-class in CSS is a keyword added to a selector that specifies a special state of the selected element.Pseudo-classes are a powerful tool in CSS that allow you to create dynamic and interactive web pages, can be used in combination with other CSS features to create complex and responsive designs.
